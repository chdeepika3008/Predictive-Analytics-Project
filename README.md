# Predictive Analytics Using Historical Data

## Project Overview
This project uses historical sales data to predict future sales trends using Linear Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Features
- Data Cleaning and Preprocessing
- Linear Regression Model
- Future Sales Forecasting
- Model Evaluation using R² Score
- Data Visualization

## Project Structure
- data/ : Dataset
- src/ : Python source code
- images/ : Output graphs
- notebooks/ : Jupyter notebooks

## Results
The model predicts future sales based on historical trends and visualizes the forecast.
